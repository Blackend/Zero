# Zero

Program rozruchowy w czystym języku asemblera dla plików "jądra systemu" działający w 16, 32 i 64 bitowym trybie procesora.

--- [PL] -----------------------------------------------------------------------

Kompilacja (GNU/Linux):

	make

Uruchomienie:

	qemu-system-x86_64 -hda build\disk.raw -m 2
